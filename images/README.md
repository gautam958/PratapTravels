## How to Add Your Images

### 1. Hero Banner
- Add your hero image as `images/hero/hero-banner.jpg`
- Recommended size: 1920x1080px or larger
- The website will automatically use this as the hero background

### 2. Route Photos
- Add route photos to `images/routes/`
- Use the filename format: `destination-name.jpg`
- The website will reference these in the routes table

### 3. Gallery Photos
- Add tourist place photos to `images/gallery/`
- Use descriptive filenames
- These will appear in the visitor dashboard gallery

### 4. Service Icons
- Add service images to `images/services/`
- Use descriptive filenames

## Supported Formats
- **JPG/JPEG** - Best for photos (smaller file size)
- **PNG** - Best for graphics with transparency
- **WebP** - Modern format, best compression

## Image Optimization Tips
1. Compress images before uploading (use tools like TinyPNG or Squoosh)
2. Keep hero images under 500KB for fast loading
3. Keep gallery images under 200KB each
4. Use consistent aspect ratios (16:9 for hero, 4:3 for gallery)

## File Naming Convention
- Use lowercase letters
- Use hyphens instead of spaces
- Be descriptive: `baba-temple.jpg` not `img1.jpg`
